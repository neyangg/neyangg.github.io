磨刀不误砍柴工系列(0)-基于CentOS/Xfce搭建最简测试环境

发表于 2016-12-29 | 更新于: 2018-12-21 | 分类于 现学现卖

前言

最近温习了一遍Linux的相关操作，为后面在这个平台上进行SQL、Python等数据分析应用做准备，作为练手，重新在虚拟机里搭建测试环境并做记录，供以后查阅。

相关软件版本：VMware Workstation11.0，CentOS7.0 minimal，Xfce4.4

系统&桌面环境选择

和很多朋友一样，我Linux的入门书籍是《鸟哥的私房菜》，所以最早接触的就是CentOS这个发行版，相对比较熟悉，而且个人比较偏好这种简单、清爽的风格，相应的作为配套，选择Xfce作为桌面环境，可以配合完成Gui下的操作，又不造成过高的系统负载。

镜像下载

因为众所周知的原因，官网下载速度很慢，推荐使用阿里云的系统镜像，点击前往，可以下载到当前所有的CentOS版本，这里选择CentOS-7.0-1406-x86_64-DVD.iso。

系统安装

CentOS最小化安装

由于是镜像安装，所以在VMware虚拟机安装配置向导页里面选择相应的iso文件，网络类型选择NAT，其他的均保持默认选择，然后点击开启虚拟机，进入图形化界面的系统安装过程。

接下来需要注意的是软件选择和系统分区，可以根据自己需求进行配置，这里我选择最小化安装和划分/boot,/,/home,/swap这几个分区（其中除swap外，其余分区均选用xfs格式）

增加管理员账户

除配置阶段外，推荐使用管理员账户进行日常操作，

$useradd -m -g neyan neyan #添加账户，同名组，并添加账户工作目录
$passwd neyan #设置密码

赋予账户管理员身份，

$visudo

在最后一行增加neyan ALL=(ALL) ALL

网络配置

首先查看局域网能否ping通，

$ping 192.168.137.1

成功，然后再看外网，

$ping www.baidu.com

失败，打开/etc/sysconfig/network-scripts/ifcfg-eno16777736文件进行网络配置，

BOOTPROTO=none #静态IP：static,动态IP：dhcp
ONBOOT=yes
IPADDR=192.168.137.2 #IP地址
NETMASK=255.255.255.0 #子网掩码
GATEWAY=192.168.137.1 #网关
DNS1=8.8.8.8 #DNS地址

重启网络服务，使配置生效，

$systemctl restart network

最后查看网络配置，CentOS在7.0版本之后默认没有内置ifconfig命令，可以自行安装（被ip取代，但因为后面操作中对ifconfig有依赖关系，所以用yum命令进行安装）

$yum install net-tools

配置yum源

添加第三方源，以.repo文件的形式出现在/etc/yum.repos.d/文件夹下，

$yum install epel-release

或者，直接修改/etc/yum.repos.d/CentOS-Base.repo文件，格式参考Base等官方源。如果引入过多的社区、第三方源，需要对其优先级顺序进行管理，

$yum install yum-priorty

插件启用/禁用设置可直接修改/etc/yum/pluginconf.d/priorities.conf，使用方法为修改/etc/yum.repos.d/CentOS-Base.repo，在base，updates，extras后面增加priority=1，在centosplus后面增加priority=2，其他后面增加priority=12。

执行系统更新

$yum update

安装X Window system

$yum groupinstall ‘X Window system’

安装xfce

$yum groupinstall xfce

进入xfce桌面环境

$systemctl isolate graphical.target

如果需要设置默认登陆环境为图形界面，

$systemctl set-default graphical.target

安装中文字体和中文输入法

新建中文字体文件夹，

$mkdir /usr/share/fonts/chinese
$cd /usr/share/fonts/
$chmod 755 chinese
$cd chinese

从windows目录C:\Windows\Fonts拷贝中文字体文件到chinese文件夹下，

$mkfontscale #生成fonts.scale文件，如果提示 mkfontscale: command not found，需自行安装，$yum install mkfontscale
$mkfontdir #生成fonts.dir文件
$fc-cache -fv #重新建立字体缓存

然后重启系统，中文字体自动生效。

输入法平台选择ibus，输入法选择ibus-libpinyin，也需要同时安装几个辅助软件，

$yum install ibus
$yum install ibus-libpinyin
$yum install ibus-gtk3 #ibus托盘显示
$yum install gtk3-immodule-xim #输入法候选字显示
$yum install im-chooser #输入法平台选择程序

分别执行im-chooser和ibus-setup选择输入法平台和输入法，重新登陆后生效。

其他

到此，基础的测试环境就搭建好了，接下来就是按自己使用需求和习惯进行软件的安装和配置。
